📚 WHAT I LEARNED
Data encoding converts data into a different format so it can be safely transmitted or stored — it is not encryption and provides no security

Base64 is one of the most common encoding schemes — it converts binary data into ASCII text using 64 printable characters

URL encoding replaces special characters with a % sign followed by their hex value so they can be safely used in web addresses

Encoding is reversible by anyone — no key is needed, which is why it must not be confused with encryption

Attackers often use encoding to disguise malicious payloads and bypass simple security filters

🔐 WHY IT MATTERS
