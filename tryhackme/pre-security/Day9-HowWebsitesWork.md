📚 WHAT I LEARNED

Websites are built with HTML for structure, CSS for styling, and JavaScript for interactivity

The browser renders HTML top to bottom, building a DOM (Document Object Model) that JavaScript can manipulate

Front-end code runs in the browser while back-end code runs on the server — both can be attack surfaces

Sensitive data should never be stored in client-side HTML or JavaScript as it is visible to anyone who views the source

JavaScript can be injected into pages if inputs are not properly sanitised — the basis of XSS attacks
