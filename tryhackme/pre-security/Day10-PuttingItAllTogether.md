📚 WHAT I LEARNED

A single web request involves multiple layers working together — DNS resolution, TCP connection, HTTP request, and server response

Load balancers distribute traffic across multiple servers to improve performance and availability

CDNs (Content Delivery Networks) cache content closer to users to reduce latency

Web application firewalls (WAFs) sit between users and servers to filter malicious requests

Databases store dynamic content and are queried by back-end code to build web pages on the fly
