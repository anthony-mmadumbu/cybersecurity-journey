📚 WHAT I LEARNED
JavaScript is the primary language of the web — it runs directly in the browser and makes pages interactive

Variables, functions, and control flow in JavaScript follow similar logic to Python but with different syntax

JavaScript can access and manipulate the DOM — the structure of a web page — allowing dynamic content changes

Because it runs client-side, JavaScript code is visible to anyone who views the page source

Malicious JavaScript injected into a page is the basis of Cross-Site Scripting (XSS) attacks

🔐 WHY IT MATTERS
