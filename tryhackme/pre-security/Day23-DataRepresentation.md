📚 WHAT I LEARNED
All data in a computer is ultimately stored as binary — sequences of 1s and 0s called bits

A byte is 8 bits and can represent 256 different values (0–255) — the building block for all data types

Hexadecimal groups 4 bits into a single character (0–9, A–F) making binary data much easier to read and work with

Colours are represented as 3 bytes — one each for red, green, and blue — giving over 16 million possible colours

Text, images, audio, and every other file type are all just different ways of organising and interpreting bytes
