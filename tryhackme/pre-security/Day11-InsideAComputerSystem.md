📚 WHAT I LEARNED

A computer is made up of core components — CPU, RAM, storage, and motherboard — each with a distinct role

The CPU executes instructions using the fetch-decode-execute cycle and its speed is measured in GHz

RAM is volatile memory used for active processes, while storage (HDD/SSD) is non-volatile and holds data long-term

The motherboard connects all components and manages communication between them via buses
Understanding hardware helps identify physical attack vectors such as cold boot attacks and hardware keyloggers
