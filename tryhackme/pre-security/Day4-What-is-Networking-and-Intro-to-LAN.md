📚 WHAT I LEARNED
A network is any group of connected devices that can communicate, using unique identifiers like IP and MAC addresses
The internet is made up of many private and public networks, and data travels between them using protocols
LAN (Local Area Network) connects devices in a small area using switches and routers
Network topologies (star, bus, ring) affect how devices communicate and how failures spread
Subnetting divides a network into smaller segments to improve performance and security
