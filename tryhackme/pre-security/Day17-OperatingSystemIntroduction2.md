📚 WHAT I LEARNED

File systems organise how data is stored and retrieved — common types include NTFS (Windows), ext4 (Linux), and APFS (macOS)

Processes have states — running, waiting, and terminated — and the OS scheduler decides which runs at any given time

User accounts and permissions control what each user can access or execute on the system
System calls are the interface through which programs request services from the OS kernel

Logs and audit trails generated by the OS are a key source of evidence during security investigations
