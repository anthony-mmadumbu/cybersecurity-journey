📚 WHAT I LEARNED

Cloud computing delivers on-demand computing resources over the internet — storage, servers, and software without owning the hardware

The three main service models are IaaS (infrastructure), PaaS (platform), and SaaS (software) — each with different levels of user control

Deployment models include public, private, and hybrid cloud — each with different security and cost trade-offs

The shared responsibility model defines what the cloud provider secures versus what the customer is responsible for

Misconfigured cloud storage (e.g. public S3 buckets) is one of the most common causes of data breaches today
