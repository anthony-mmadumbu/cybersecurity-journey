Windows uses the NTFS file system which supports permissions, encryption, and detailed access control lists (ACLs)

Task Manager and Services allow monitoring of running processes and background services on the system

User Account Control (UAC) prompts users before allowing actions that require elevated privileges

Windows Event Viewer logs system, security, and application events — a critical tool for investigating incidents
