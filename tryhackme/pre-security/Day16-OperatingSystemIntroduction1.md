📚 WHAT I LEARNED
An operating system manages hardware resources and provides services for applications to run on top of The kernel is the core of the OS — it handles communication between software and hardware directly

User mode and kernel mode separate regular application processes from privileged OS operations for security

The OS manages processes, memory, file systems, and I/O devices through structured abstractions
Common OS families include Windows, Linux, and macOS — each with different architectures and security models
