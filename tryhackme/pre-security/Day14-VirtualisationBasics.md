📚 WHAT I LEARNED

Virtualisation allows multiple virtual machines (VMs) to run on a single physical host, sharing its hardware resources

A hypervisor manages VMs — Type 1 runs directly on hardware (bare-metal), Type 2 runs on top of an existing OS

VMs are isolated from each other, making them useful for testing malware and running labs safely

Snapshots allow a VM to be saved and restored to a previous state, which is valuable in security testing

Containers (like Docker) are a lighter form of virtualisation that share the host OS kernel
