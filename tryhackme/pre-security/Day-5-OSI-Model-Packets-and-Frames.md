📚 WHAT I LEARNED

The OSI model has 7 layers — each responsible for a specific part of how data travels across a network

Data is encapsulated as it moves down the layers and de-encapsulated as it moves back up on the receiving end

Packets carry data across networks with headers containing source/destination IP addresses
Frames operate at Layer 2 and use MAC addresses to deliver data within a local network

TCP and UDP handle data differently — TCP is reliable and connection-based, UDP is faster but has no guarantee of delivery

🔐 WHY IT MATTERS

The OSI model is a universal reference — understanding it helps diagnose where in a network something is going wrong

Attackers often target specific layers (e.g. Layer 3 for IP spoofing, Layer 7 for application attacks) — knowing the model helps identify the right defence
